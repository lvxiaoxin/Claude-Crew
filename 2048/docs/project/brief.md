# Project Brief

## Original Idea
I want to build a 2048 game with electron thus I can play on Mac. The most important things, great UI, great animation.

## Structured Summary
- **What**: A beautifully designed 2048 puzzle game as a native Mac desktop app
- **Who**: The user (Mac desktop gamer)
- **Why**: Play 2048 natively on Mac with premium visual experience
- **Platforms**: macOS (Electron)
- **Key Features**:
  - Classic 2048 gameplay mechanics
  - Great UI design with polished look and feel
  - Smooth, satisfying animations (tile sliding, merging, appearing)
  - Native Mac app experience via Electron

## Constraints & Preferences
- Must run on macOS via Electron
- UI quality and animation are the TOP priorities
- Should feel like a premium native app
